Who

This is a side project realized by TrinTragula (@TrinTragula) in his spare time.

How

This website simply scans the provided website for any fonts it may have in its pages using some simple but fancy logic. Feel free to use it to download any cool font you may find online, but please be sure to only use it in your own projects if you legally own it. I take no responsibility for the use you may make of the font you download.

Why

I often found myself wondering which font a website was using and I was tired of always having to open up the developer console.

Do you collect data?

Nope, I just collect some anonymous usage statistcs to see if someone is actually using this website or not.

Why do I see "Font not loaded" instead of a preview of the font?

Some font types do not work well with the preview, or maybe it's just a false positive I found which is not actually a font.
